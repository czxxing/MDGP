# 多模态数据管理平台

这是一个基于Streamlit的多模态数据管理平台，支持对图像、视频、音频、文本等多种格式文件的管理和统计分析。

## 项目结构

项目采用模块化设计，分为三个主要模块：

```
MDGP/
├── multimodal_processor/     # 多模态文件处理模块
│   ├── __init__.py
│   └── file_processor.py     # 文件扫描、类型识别、统计功能
├── lance_db/                 # Lance数据库操作模块
│   ├── __init__.py
│   └── lance_manager.py      # Lance数据库管理类
├── streamlit_ui/            # Streamlit界面模块
│   ├── __init__.py
│   └── ui_components.py      # 界面组件和布局
├── app.py                   # 主应用入口
├── requirements.txt         # 依赖包列表
├── data/                    # 数据目录（存放多模态文件）
├── db/                      # 数据库目录（存放Lance格式数据）
└── .gitignore              # Git忽略文件
```

## 模块功能说明

### 1. multimodal_processor 模块
- **文件类型识别**：自动识别图像、视频、音频、文本等文件类型
- **目录扫描**：递归扫描指定目录，收集文件信息
- **统计分析**：生成文件数量、大小、类型分布等统计信息

### 2. lance_db 模块
- **数据存储**：将文件信息保存到Lance格式数据库
- **数据加载**：从Lance数据库读取数据到DataFrame
- **数据导出**：支持CSV、JSON、Parquet格式导出
- **数据库管理**：提供数据库信息查询和清理功能

### 3. streamlit_ui 模块
- **界面布局**：创建三选项卡布局（数据目录、数据处理、数据统计）
- **交互组件**：按钮、选择框、数据表格等交互元素
- **可视化**：文件类型分布饼图、大小分布柱状图

## 使用方法

1. 安装依赖：
```bash
pip install -r requirements.txt
```

2. 将多模态文件放入 `data/` 目录

3. 运行应用：
```bash
streamlit run app.py
```

## 功能特性

- **多模态支持**：支持图像、视频、音频、文本等多种文件格式
- **高效存储**：使用Lance格式进行高效数据存储和查询
- **可视化统计**：提供文件类型分布和大小分布的可视化图表
- **数据导出**：支持多种格式的数据导出功能
- **模块化设计**：代码结构清晰，易于维护和扩展

## 技术栈

- **前端框架**：Streamlit
- **数据处理**：Pandas, Daft
- **数据库**：Lance, PyArrow
- **可视化**：Matplotlib
- **文件处理**：PIL, NumPy
